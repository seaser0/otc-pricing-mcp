# otc-pricing-mcp

An open-source MCP (Model Context Protocol) server for the Open Telekom Cloud Price Calculator API.

**Status**: Under development (v0.1.0)

## Overview

This server exposes the OTC Price Calculator as an MCP-compatible service, enabling LLM clients to query pricing for OTC services across multiple regions.

## Quick Start

(Coming in v0.1.0)

## Features

- 7 MCP tools for service discovery, pricing queries, and cost estimation
- Support for multiple OTC regions (eu-de, eu-nl, eu-ch2)
- Currency-aware pricing (EUR)
- Multi-service queries with automatic fan-out
